LumberJack 
Data log control for Saleae Logic
1) Make sure you are using the latest version of the Saleae Software, 1.2.5 as of this writing.
2) Make sure to enable “scripting socket server” 
Options>Preferences>Developer-> make sure the Enable scripting socket server is ticked.
3) Run Setup which is found in the “publish” directory on the GitHub repository.
4) Start the Saleae Logic 1.2.5 program.
5) Set the monitor to your needs, being sure that “scripting socket server” is enabled.
6) Launch LumberJack and follow the prompts, using the defaults should create a directory “C:/traces/” for your datalogs.
7) You can review your traces with Options>Open capture / setup (or CTRL O)
